# Multiple Indicator Multiple Cause Model of location on Eroticism
# Lavaan syntax
```
MIMIC_Ero_model <-'
# measurement model
 Eroticism =~ M_Ero + X_Ero + F_Ero

# regressions 
 Eroticism ~ Inactive + Education + Work + Gym + Social

# (residual) (co)variances 
 Inactive ~~ Education
 Inactive ~~ Gym
 Inactive ~~ Work
 Inactive ~~ Social
 Education ~~ Gym
 Education ~~ Work
 Education ~~ Social
 Gym ~~ Work
 Gym ~~ Social
 Work ~~ Social
'
```

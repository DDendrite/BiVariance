# Multiple Indicator Multiple Cause Model of location on Attraction
# Lavaan syntax
```
MIMIC_Atr_model <-'
# measurement model
 Attraction =~ M_Atr + X_Atr + F_Atr

# regressions 
 Attraction ~ Inactive + Education + Work + Gym + Social

# (residual) (co)variances 
 Inactive ~~ Education
 Inactive ~~ Gym
 Inactive ~~ Work
 Inactive ~~ Social
 Education ~~ Gym
 Education ~~ Work
 Education ~~ Social
 Gym ~~ Work
 Gym ~~ Social
 Work ~~ Social
'
```
